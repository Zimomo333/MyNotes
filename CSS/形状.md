# 0.5px线

1. 采用transform: scale()的方式

```css
#line {
    border-top: 1px solid;
    width: 300px;
    height: 300px;
    transform: scale(0.5);
}
```

2. 采用meta viewport的方式

```
<meta name="viewport" content="width=device-width, initial-scale=0.5, minimum-scale=0.5, maximum-scale=0.5, user-scalable=no"/>
```
